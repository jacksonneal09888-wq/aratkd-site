Admin Quick Reference

  • ✅ Sidebar remembers your last active tab across sessions (localStorage).
  • ✅ Events marked Active appear on the student kiosk check-in screen.
  • ✅ Archiving a student removes them from the live roster for 30 days, then permanently deletes.
  • ✅ Attendance count only resets to 0 when a student is promoted to a new belt rank.
  • ✅ Students can earn more lessons than required — the threshold is a minimum, not a cap.
  • ✅ Kiosk week theme is pulled live from the Google Sheets calendar — no code deploy needed when the calendar updates.
  • ✅ The kiosk only accepts check-ins on Mon / Wed / Fri.
